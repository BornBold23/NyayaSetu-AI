# Functional Requirements

## Citizen

- Register/Login
- Search crime cases
- Browse crime awareness library
- Submit anonymous reports
- Upload evidence
- View crime statistics
- Read AI summaries

---

## Moderator

- Review community reports
- Approve or reject reports
- Verify evidence
- Manage reported cases

---

## Administrator

- Manage users
- Manage cases
- Update case status
- Upload awareness articles
- Generate reports
- View dashboard analytics

---

## AI Module

- Generate case summaries
- Classify crime categories
- Recommend safety tips
- Extract important information

---

# Non Functional Requirements

- Secure Login
- Fast Search
- Responsive UI
- Dark Theme
- Easy Navigation
- Mobile Friendly
