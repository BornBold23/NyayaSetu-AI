
# 🇮🇳 NyayaSetu AI

## AI-Powered Crime Awareness, Community Reporting & Case Intelligence Platform for India

NyayaSetu AI is a Data Science and AI-powered platform designed to promote crime awareness, community participation, and intelligent case analysis across India.

The platform enables citizens to anonymously report incidents, explore famous criminal cases, analyze crime trends, and access AI-generated case summaries through an interactive dashboard.

---

## 🚀 Features

- 📚 Crime Awareness Library
- 🔍 Search Criminal Cases
- 📝 Anonymous Community Reporting
- 🤖 AI Case Summarization
- 📊 Crime Analytics Dashboard
- 📍 State-wise Crime Statistics
- ⚖️ Case Status Tracking
- 🚨 Safety Awareness & Emergency Guidance
